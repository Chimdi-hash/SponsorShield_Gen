How Strict Consensus Works

  1. 01 Each validator independently renders the proof URL in a headless browser.
  2. 02 The HTML is passed to the LLM with the sponsor's rules.
  3. 03 "APPROVED" or "REJECTED" — all must match or deviant validators are slashed.
  4. 04 Consensus is finalised on-chain and payout released instantly.
